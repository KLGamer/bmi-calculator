# Body Mass Index (BMI) Calculator  
This Python-based **BMI Calculator** is a practical tool designed to quantify body mass index by processing **user-inputted weight and height**. Utilizing conditional logic, it categorizes results into health tiers like **underweight, normal, or obese**. It serves as an excellent introductory project for mastering data types, arithmetic operators, and user input.

### Key Features of the Project
* **Dynamic Input:** Accepts weight (kg) and height (m) directly from the user.
* **Precision Calculation:** Uses the standard formula: $BMI = \frac{mass_{kg}}{height_{m}^2}$.
* **Health Insights:** Provides instant feedback based on official BMI classifications.

### Setup
1. **Install Python:** Ensure Python 3.x is installed on your system.
2. **Save Script:** Copy the code into a file named `bmi_calc.py`.
3. **Run:** Open your terminal and type `python bmi_calc.py`.

### Usage
1. **Input Weight:** Enter your mass in kilograms (e.g., 70).
2. **Input Height:** Enter your height in meters (e.g., 1.75).
3. **Review Results:** The script will output your BMI value and health status.
